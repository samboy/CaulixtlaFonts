The Caulixtla Font set

This is a set of six different fonts (three type faces) which I use on my personal webpage:
  • Serif regular
  • Serif italic
  • Serif bold
  • Sans semi-bold
  • Sans semi-bold Small Caps
  • Monospace
In order to minimum page load times, the primary fonts (Serif regular, Serif italic, and Sans semi-bold) have been subsetted to have only symbols in English and Spanish:

The fonts have been converted in to all modern web font formats except for the short-lived and obsolete SVG font format: WOFF, WOFF2, and even the now-obsolete EOT used in mid-first-2000s-decade versions of Internet Explorer. In addition, the Fontforge .sfd source files are available for the primary fonts (Again: Serif regular, Serif italic, and Sans semi-bold).

Files can be viewed at the GitHub repo. There’s also a sample of how text looks rendered with the primary fonts.

The fonts are open source, as described in COPYING.TXT.
